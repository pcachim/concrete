## Material properties

Concrete

- [Basic properties](./ec2-material.ipynb)
- [Creep](./ec2-fluencia.ipynb)
- Shrinkage

Steel

- Reinforcing steel
- Prestressing steel

## JSON database file

A [database](./eurocodes.json) containing information regarding the properties of Eurocode materials, loads and sections.
